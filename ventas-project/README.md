#  Registro de Ventas Frontend

Aplicación web para registrar ventas, calcular totales y mostrar resúmenes de datos.  
Construida con **React + TypeScript + Vite**.

---

##  Instalación y configuración

1. # clonar repositorio
   ```bash
   git clone https://github.com/TOMAS-PENROZ-M/Empresariales.git
   cd ventas-project

2. # instalar dependecias
    npm install

3. # lenvatar servidor 
    npm run dev
